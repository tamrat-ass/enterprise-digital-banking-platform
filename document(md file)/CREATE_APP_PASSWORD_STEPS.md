# 📱 How to Create Gmail App Password - Step by Step

## Step 1: Go to Gmail Security Settings

1. Open your browser
2. Type this URL in address bar:
   ```
   https://myaccount.google.com/security
   ```
3. Press Enter
4. Sign in if asked

**You should see a page that says "Google Account"**

---

## Step 2: Find "2-Step Verification"

On the left side of the page, look for:
```
How you sign in to Google
```

Under this section, you should see:
```
✅ 2-Step Verification
```

**IMPORTANT: It must say "ON" (enabled)**

If it says "OFF":
1. Click on "2-Step Verification"
2. Follow Google's steps to enable it
3. You'll need your phone to verify
4. Then come back to this guide

---

## Step 3: Go to App Passwords Page

Once 2-Step Verification is ON:

1. Go to this URL:
   ```
   https://myaccount.google.com/apppasswords
   ```

2. Press Enter

**You should now see a page with two dropdown menus**

---

## Step 4: Select "Mail"

Look for first dropdown that says:
```
Select the app
```

1. Click on this dropdown
2. Find and click: **Mail**

**You should now see it says "Mail" in the first dropdown**

---

## Step 5: Select "Windows Computer"

Look for second dropdown that says:
```
Select the device
```

1. Click on this dropdown
2. Find and click: **Windows Computer**
   - (If you're on Mac, select "Mac")
   - (If you're on phone, select "iPhone" or "Android")

**You should now see "Windows Computer" in the second dropdown**

---

## Step 6: Click Generate

1. Look for a button that says: **Generate**
2. Click it

**Wait a moment...**

---

## Step 7: Copy Your App Password

Google will show a yellow box with your 16-character password

**It looks like:**
```
abcd efgh ijkl mnop
```

This is your APP PASSWORD!

**IMPORTANT:**
- ✅ Copy it exactly as shown
- ✅ It has spaces between groups
- ✅ It's 16 characters total
- ✅ It's lowercase letters only

---

## Step 8: Send Me The Password

Once you have the 16-character password:

**Reply with:**
```
App Password: abcd efgh ijkl mnop
```

**Example:**
```
App Password: xyzw abcd efgh ijkl
```

## Troubleshooting

### Issue: Can't find 2-Step Verification

**Solution:**
1. Make sure you're at: https://myaccount.google.com/security
2. Look in the left menu
3. Look for "How you sign in to Google"
4. You should see "2-Step Verification" option

### Issue: 2-Step Verification is OFF

**Solution:**
1. Click on "2-Step Verification"
2. Click "Start setup" or "Enable"
3. Follow Google's instructions
4. You'll need your phone to verify
5. Come back to this guide when done

### Issue: Can't find "App passwords" option

**Solution:**
1. Make sure 2-Step Verification is ON first (critical!)
2. Then go to: https://myaccount.google.com/apppasswords
3. If you still don't see it:
   - Try a different browser
   - Try private/incognito mode
   - Sign out and sign back in

### Issue: Only see "Select the app" dropdown, not "Select the device"

**Solution:**
1. Make sure you clicked on the first dropdown
2. Select "Mail"
3. Then second dropdown should appear
4. Select your device
5. Then click Generate
