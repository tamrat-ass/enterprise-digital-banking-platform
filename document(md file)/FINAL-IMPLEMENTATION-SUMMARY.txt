================================================================================
                          API ENDPOINT
================================================================================

POST /api/users/toggle-status

Request:
  {
    "userId": "user_9afe6a178cc20743d123d660",
    "newStatus": "disabled"  // Optional
  }

Response (200):
  {
    "success": true,
    "message": "User status changed to disabled",
    "userId": "user_9afe6a178cc20743d123d660",
    "newStatus": "disabled",
    "previousStatus": "active"
  }

Using Curl:
  curl -X POST http://localhost:3000/api/users/toggle-status \
    -H "Content-Type: application/json" \
    -d '{"userId":"user_id","newStatus":"disabled"}'
