🐛 TROUBLESHOOTING QUICK HELP:
================================================================================

Problem: "Database connection refused"
Solution: Verify PostgreSQL is running and DATABASE_URL is correct

Problem: "No data in database"
Solution: Run: pnpm db:populate

Problem: "API returns no data"
Solution: Make sure app is running with: pnpm dev

Problem: "Seeding script fails"
Solution: Install pg library: npm install pg

Solution: Run: pnpm db:populate
