                            WHAT WAS DONE
================================================================================

Modified File:

Change Made:
  Added "emailVerified: true" to user update query

Before:
  await db.update(user).set({
    passwordHash: passwordHash,
    passwordChangedAt: new Date(),
    updatedAt: new Date(),
  })

After:
  await db.update(user).set({
    passwordHash: passwordHash,
    emailVerified: true,  ← NEW: Auto-verify email
    passwordChangedAt: new Date(),
    updatedAt: new Date(),
  })

================================================================================
                            HOW IT WORKS
================================================================================

Password Reset Flow (Updated):
  1. Admin clicks "Reset Password" button on user
  2. User is sent to /set-new-password page
  3. User enters strong password (validation)
  4. Submit password
  5. API receives POST /api/users/reset-password
  6. API validates password (8+ chars, complexity)
  7. API updates database:
     ✅ Sets new password hash
     ✅ Sets emailVerified = true  ← NEW AUTOMATIC
     ✅ Updates timestamps
  8. Success response (200)
  9. User redirected to dashboard
  10. User can now sign in immediately!

Database Update:
  UPDATE "user"
  SET 
    passwordHash = $1,           -- New hashed password
    emailVerified = true,        -- ← AUTO-SET
    passwordChangedAt = NOW(),
    updatedAt = NOW()
  WHERE id = $2
