### System
- **6 roles** with hierarchy
- **10-25 permissions** per role
- **6 core database tables**
- **1 optimized query** for permissions
- **Single session per user**

### Permissions
- **Format**: `module.action` (e.g., `users.create`)
- **Modules**: 15+ (users, documents, approvals, etc.)
- **Actions**: 10+ (view, create, edit, delete, approve, etc.)

### Roles
1. Super Admin (Level 100)
2. Executive (Level 90)
3. Compliance Officer (Level 70)
4. Auditor (Level 60)
5. Department Head (Level 50)
6. Staff (Level 10)

### API Endpoints
- **POST** `/api/custom-signin` - Login
- **GET** `/api/users/me` - Current user
- **POST** `/api/users` - Create user
- **PUT** `/api/users/[id]` - Update user
- **DELETE** `/api/users/[id]` - Delete user
