USER WORKFLOW
═══════════════════════════════════════════════════════════════════════════════

To Delete a File:
  1. Go to File Management
  2. Find file
  3. Click trash icon
  4. Confirm in dialog
  5. File moves to Recycle Bin ✅

To View Recycle Bin:
  1. Click "Recycle Bin" in sidebar ✅
  2. See all deleted files ✅
  3. Search, sort, filter as needed ✅

To Restore a File:
  1. Find deleted file in Recycle Bin
  2. Click restore icon (green)
  3. Confirm in dialog
  4. File returns to File Management ✅

To Permanently Delete:
  1. Find deleted file in Recycle Bin
  2. Click delete icon (red)
  3. Confirm warning dialog
  4. File permanently deleted ✅

To Bulk Restore:
  1. Select multiple files with checkboxes
  2. Click "Restore X" button
  3. Confirm in dialog
  4. All files restored ✅

To Bulk Delete:
  1. Select multiple files
  2. Click "Delete X" button
  3. Confirm warning dialog
  4. All files permanently deleted ✅
