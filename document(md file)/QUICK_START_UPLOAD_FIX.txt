QUICK START - FILE UPLOAD FIX
================================================================================

PROBLEM: "This document does not have a file attached"
ROOT CAUSE: file_path is NULL in database

================================================================================
3-MINUTE QUICK FIX
================================================================================

1. Check Database:
   Open: http://localhost:3000/api/admin/fix-database
   Expected: {"success": true, "action": "exists"}

2. Check Status:
   Open: http://localhost:3000/api/admin/test-upload
   Look for: "documentsWithFilePath" count

3. Upload Test:
   Go to: http://localhost:3000/upload
   Upload small file (test.txt or small PDF)
   Watch: Terminal logs for [FileStorageService] and [DocumentService]
