HOW EMAILS ARE SENT:

Development Mode (Current):
   1. Email triggered (user creation, password reset, etc.)

      • To address
      • From address
      • Subject
      • Timestamp
      • HTML preview
   4. User can see all details in server logs

Production Mode (When Deployed):
   1. Email triggered
   2. System detects: NODE_ENV=production
   3. Attempts to send via SendGrid API
   4. If verified domain configured:
