1. INSTALL DEPENDENCIES

2. INITIALIZE DATABASE (Create all tables)
   
   After dependencies are installed, run:
   
   > pnpm db:init
   
   This will:
   - Connect to your PostgreSQL database
   - Create all 18 tables
   - Create indexes for performance
   - Show verification

   Expected: "✅ Total tables created: 18"


3. START THE APPLICATION
   
   After database is initialized, run:
   
   > pnpm dev
   
   This starts:
   - Frontend: React 19 UI
   - Backend: Next.js API (31 endpoints)
   - Live reload on code changes

   Expected: "✓ Ready in X.Xs" and "Local: http://localhost:3000"


4. OPEN IN BROWSER
   
   Go to: http://localhost:3000
   
   You should see the Meridian login page.


5. CREATE FIRST USER (SUPER ADMIN)
   
   - Click "Sign up"
   - Fill in email, password, name
   - Select any role (will be upgraded to Super Admin)
   - Click "Create account"
   - Login with your credentials


6. START EXPLORING
   
   After login, you can:
   - View Dashboard (see metrics)
   - Create Documents
   - Create Projects
   - Create Vendors
   - Create Contracts
   - Manage Risks
   - Track Compliance
   - View Analytics

================================================================================
                    TROUBLESHOOTING
================================================================================

Issue: "Cannot connect to database"
Fix: Verify PostgreSQL is running and database exists
  psql -U postgres -d ahadufile -c "SELECT 1;"

Issue: "Port 3000 already in use"
Fix: Use different port
  PORT=3001 pnpm dev

Issue: "Module not found"
Fix: Reinstall dependencies
  rm -rf node_modules
  pnpm install

Issue: "Tables already exist"
Fix: This is fine, just continue with pnpm dev
